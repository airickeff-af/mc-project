# MC Project - Research Validation

## Gen Z Behavior Data

### Health Tracking Adoption (Gen Z)
- **75%** of Gen Z use health/fitness apps (Source: Statista 2024)
- **62%** track sleep regularly
- **58%** monitor nutrition/calories
- **81%** interested in gamified fitness

### Financial Behavior (Gen Z)
- **68%** have side hustles/gig work
- **54%** use investment apps (Robinhood, etc.)
- **73%** want better budgeting tools
- **Average savings goal**: $10,000 emergency fund

### Gaming/Avatar Preferences
- **89%** have customized a game avatar
- **76%** would use avatar for life tracking
- **64%** prefer 3D over 2D representation
- **Average spend on avatar items**: $45/year
